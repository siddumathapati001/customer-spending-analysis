# Customer Spending Analysis

## Problem Statement
Analyze customer spending patterns to identify key drivers of average monthly spend and provide business insights.

## Dataset
Customer-level transactional and demographic data used for exploratory data analysis and statistical evaluation.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and missing value treatment
- Exploratory data analysis (EDA)
- Statistical comparison of spending patterns
- Visualizations to support insights
